### **_Description_**


TasteTogether is a web app where food lovers can share recipes, discover new dishes, and connect with friends. You can post photos of your homemade meals or restaurant finds, chat with friends via video calls while cooking or eating, and even watch shows together. ### **_Technologies Used_**

- Node.js
- Express.js
- MongoDB + Mongoose
- JavaScript / HTML / CSS
- React
- Authentication & Session Middleware

## **_Special Features_**

1. User authentication and profile management
2. Share and interact with food posts
3. Video calls to cook or eat together
4. Random dish picker based on preferences
5. Random Restaurant picker based on preferences
6. Recipe and restaurant discovery using APIs
